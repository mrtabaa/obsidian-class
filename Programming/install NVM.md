Check [NVM latest version](https://github.com/nvm-sh/nvm/releases)

e.g. Installing NVM version 0.39.3
```bash
curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.39.4/install.sh | bash
```

Reset terminal
```bash
source ~/.bashrc
```

check version
```bash
nvm --version
```
